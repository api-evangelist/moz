# MozThis is a repo for managing machine readable artifacts for the [Moz APIs](http://www.seomoz.org). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Moz APIs](http://www.seomoz.org).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.seomoz.org)- [Developer](http://www.seomoz.org)- [Twitter](https://twitter.com/seomoz)- [GitHub](https://github.com/seomoz)The [APIs.json](https://github.com/api-evangelist/moz/blob/master/apis.json) index provides links to all the relevant links for the Moz APIs, including the OpenAPI and Postman if they are present.
